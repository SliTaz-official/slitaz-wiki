====== Old Cookbook ======

The Old Cookbook brings together deprecated information about the project management, operation and development of the distribution. It talks about creating packages, receipts, the wok, and scripts that start SliTaz.

==== Table of contents ====

    * [[en:oldcookbook:receipt|Receipts]]  - Receipts (recipes) for creating (cooking) SliTaz packages using Tazwok.
    * [[en:oldcookbook:wok|Wok]]  - Creating SliTaz packages from source using Tazwok. (Uses a receipt).
